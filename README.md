## Road Accident Analysis in India — Excel Dashboard Project

This project presents an interactive data analytics dashboard built in Microsoft Excel to analyze road accident patterns across India.
The objective is to transform raw accident data into clear, decision-ready insights that highlight trends, risk factors, and regional impact.

### Key Insights

* Over 15,000 casualties recorded in the dataset
* Certain states show consistently higher accident severity
* Weather conditions and alcohol involvement significantly influence accident outcomes
* Year-wise trends indicate fluctuations useful for planning and prevention

### Tools & Techniques Used

* Data cleaning and preprocessing in Excel
* Pivot Tables, Pivot Charts, and Slicers for interactivity
* KPI metric design for quick insight consumption
* Dashboard layout focused on usability and storytelling

### Purpose of the Project

To demonstrate practical **data analytics, visualization, and business insight generation** using real-world style data, while building a strong foundation for entry-level Data Analyst roles.
